# Restaurant Ordering App

A modern restaurant ordering website built using React and Vite.

## Features

- Responsive UI
- Menu Browsing
- Add to Cart
- Quantity Management
- Checkout Page
- Reservation Page
- Contact Page
- Gallery Page

## Tech Stack

- React
- Vite
- React Router DOM
- Context API
- CSS
